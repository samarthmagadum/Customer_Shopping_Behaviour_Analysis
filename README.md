# Customer Shopping Behaviour Analysis

An end-to-end data analytics project that processes raw e-commerce transaction data, uncovers business trends using SQL, and builds an interactive dashboard to optimize revenue and marketing strategies.

---

## 📌 Project Overview
This project analyzes transactional data from 3,900 customer purchases across multiple product categories. The objective is to evaluate spending patterns, subscription growth, discount impacts, and customer segmentation to deliver data-driven business recommendations.

## 📊 Dataset Summary
* **Dataset Size:** 3,900 rows and 18 columns.
* **Demographics:** Age, Gender, Location, Subscription Status.
* **Purchase Details:** Item, Category, Purchase Amount, Season, Size, Color.
* **Behavioral Data:** Discounts, Promo Codes, Frequency, Review Ratings, Shipping.

## 🛠️ Tech Stack & Tools
* **Data Cleaning & EDA:** Python (Pandas)
* **Database & Querying:** PostgreSQL / MySQL / SQL Server
* **Data Visualization:** Power BI
* **Presentations & Reports:** Gamma App (AI PPT Generation), Markdown

---

## ⚙️ Project Workflow

### 1. Data Cleaning & Exploratory Data Analysis (Python)
* Conducted initial data auditing using `.info()` and `.describe()`.
* Handled missing data by imputing 37 null `Review Rating` values using category-specific medians.
* Standardized column naming structures to clean `snake_case` syntax.
* Engineered features: `age_group` bins and `purchase_frequency_days`.
* Integrated Python with the database server to load the cleaned data.

### 2. Business Logic & Deep-Dive Querying (SQL)
Designed and executed structured database queries to solve 10 key business objectives, including:
* **Revenue Drivers:** Compared total revenue generated by male vs. female cohorts.
* **Customer Segments:** Categorized shoppers into New, Returning, and Loyal groups based on purchase frequency.
* **Discount Elasticity:** Identified products with the highest reliance on active active promos.

### 3. Executive Dashboarding & Presentations (Power BI & Gamma)
* Built a dynamic Power BI dashboard mapping Key Performance Indicators (KPIs).
* Modeled revenue splits across demographic groups, product categories, and shipping choices.
* Synthesized core insights into an executive-ready slide deck using Gamma.

---

## 📈 Key Results & Insights
* **High-Value Demographics:** "Young Adults" represent the highest-revenue age cohort ($62,143).
* **Gender Spend Dynamics:** Male customers drive the majority of top-line revenue ($157,890 vs. $75,191).
* **Loyalty Insights:** Over 79% of the customer base (3,116 shoppers) fall into the "Loyal" segment.
* **Subscription Reality:** Subscribers ($59.49) and non-subscribers ($59.87) maintain nearly identical average order values, highlighting an opportunity to upsell premium subscriber benefits.
